# Project Group 12 ‚Äî Mom's Coders - Furniture - –ú–µ–±–ª–µ—Ä—ñ—è 

**Project Group 12** ‚Äî a team project developed as part of a study program.  
The goal was to create a modern, responsive furniture gallery website with interactive elements, dynamic content, and modal windows.

# Project Description

The website consists of the following main sections:

1. **Header** ‚Äî the top navigation menu with a logo and links, enabling smooth scrolling to different sections.  
2. **Hero** ‚Äî a short introductory block with a large button that, when clicked, smoothly scrolls down to the **Feedback** section.  
3. **Our Furniture (Furniture)** ‚Äî a product catalog that dynamically loads items from the backend when a category is selected (for example, *tables*).  
   - Clicking on a product card opens a modal with detailed information, an option to choose a color, and a button to proceed to the order form.  
4. **Order Form** ‚Äî a modal window where the user can fill in their details and submit an order.  
5. **About Us** ‚Äî a short section with information about the company and the purpose of the website.  
6. **FAQ** ‚Äî a ‚ÄúFrequently Asked Questions‚Äù section with an accordion for easy browsing.  
7. **Feedback** ‚Äî a client reviews section that fetches and confirms feedback from the backend.  
8. **Footer** ‚Äî a bottom section with social media links and quick navigation.

üí° **Responsiveness**: The website is fully optimized for **mobile devices**, **tablets**, and **desktop** screens.
